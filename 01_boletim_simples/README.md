# Sistema de Boletim Simples

Projeto inicial em Python para praticar estruturas sequenciais, condicionais e de repetição.

## Funcionalidades

- Cadastro do nome do aluno
- Entrada de quatro notas
- Cálculo da média
- Exibição da situação final
- Repetição do processo para novos alunos

## Conceitos praticados

- Entrada e saída de dados
- Variáveis
- Operadores aritméticos
- Estruturas condicionais
- Estruturas de repetição
